# Button Counter

This project advances digital input handling by integrating tracking variables into the control loop. Instead of directly mirroring the instantaneous hardware state, it increments an internal numeric counter variable each time a push-button press event is registered, building a foundation for state machines.

## ⚙️ Hardware & Configuration
- **MCU:** STM32F407VGT6 (ARM Cortex-M4)
- **External Components:** 1x Push-Button, 1x Pull-Down Resistor
- **Active Pin:** `PD2` (Digital Input from Button)
- **Method:** Edge Detection via Polling Loop Counter

## 🔍 Key Concepts Covered
- **State Tracking Variables:** Introducing RAM variables (`uint32_t buttonCount`) to retain system state information indefinitely across infinite loops.
- **Edge Transition Logic:** Understanding how to catch the specific moment a button transitions from open to closed, preventing the counter from running away.
- **Software Debounce Windows:** Utilizing short strategic delays to ignore rapid spurious electrical noise generated by mechanical switch contacts grinding together.

## 💻 Complete Source Code (`main.c` & GPIO Init)

Below is the complete hardware control logic and the peripheral configuration for the button counter implementation:

```c
/* GPIO Initialization Function */
static void MX_GPIO_Init(void)
{
  GPIO_InitTypeDef GPIO_InitStruct = {0};

  /* GPIO Ports Clock Enable */
  __HAL_RCC_GPIOH_CLK_ENABLE();
  __HAL_RCC_GPIOD_CLK_ENABLE();

  /* Configure GPIO pin : PD2 (External Button Input) */
  GPIO_InitStruct.Pin = GPIO_PIN_2;
  GPIO_InitStruct.Mode = GPIO_MODE_INPUT;
  GPIO_InitStruct.Pull = GPIO_NOPULL; // Externally pulled down via hardware resistor
  HAL_GPIO_Init(GPIOD, &GPIO_InitStruct);
}

/* Infinite Loop inside main() */
int main(void)
{
  /* Reset of all peripherals, Initializes the Flash interface and the Systick. */
  HAL_Init();

  /* Configure the system clock */
  SystemClock_Config();

  /* Initialize all configured peripherals */
  MX_GPIO_Init();

  uint32_t buttonCount = 0;   // Variable to store button press counts
  uint8_t buttonState = 0;    // Variable to track the current button state

  /* USER CODE BEGIN WHILE */
  while (1)
  {
    /* Read the current state of the button */
    if (HAL_GPIO_ReadPin(GPIOD, GPIO_PIN_2) == GPIO_PIN_SET)
    {
      /* Check if the button was previously NOT pressed (Edge detection) */
      if (buttonState == 0)
      {
        buttonCount++;      // Increment the counter
        buttonState = 1;    // Update state to prevent multiple increments on a single press
        
        /* Software debounce delay */
        HAL_Delay(50);
      }
    }
    else
    {
      /* Reset state when the button is released */
      buttonState = 0;
    }

    /* USER CODE END WHILE */
  }
}
